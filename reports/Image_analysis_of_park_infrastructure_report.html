1 Executive Summary

BC Parks manages a large inventory of infrastructure assets such as bridges, stairs, boardwalks, and viewing platforms. While many of these assets have associated photographs, important attributes such as material, size, structure position, railing presence, and number of steps are often missing or incomplete. This limits the organization’s ability to plan maintenance, estimate costs, and make reliable, data-driven decisions.

This project proposes the development of a computer vision pipeline to automatically extract missing infrastructure attributes from images. We begin with a zero-shot vision-language model (Radford et al. 2021) as a simple baseline to quickly assess feasibility. We then move toward a more structured approach using attribute-specific models to improve prediction accuracy for different tasks such as material classification, structural feature detection, and size estimation.

The final output will be a reproducible workflow that processes images and generates structured attribute data, along with confidence scores to support review. By reducing manual effort and improving data completeness, this solution aims to make the infrastructure dataset more reliable and useful for BC Parks operations.

1.1 Introduction

1.1.1 Problem Description

BC Parks manages a large number of infrastructure assets that support public recreation. These assets require accurate and complete data for effective maintenance and planning. However, many records in the current system are incomplete, even though images of the infrastructure are available. Important details such as material type, dimensions, and structural features are often missing.

1.1.2 Importance of the Problem

Incomplete data creates challenges for staff who rely on this information to make decisions. Manual review of images or field inspections are often required to fill in missing attributes, which is time-consuming and not scalable. Improving data completeness would allow BC Parks to better plan maintenance, reduce costs, and ensure safety across its infrastructure.

1.1.3 Background

The dataset contains thousands of infrastructure images along with partial attribute data. Previous work has not fully utilized image data to extract useful information. Advances in computer vision and machine learning, including vision-language models, provide an opportunity to automate this process and improve data quality (Radford et al. 2021).

1.1.4 Objectives

The main objective of this project is to develop a system that can automatically predict missing infrastructure attributes from images. Specifically, the project aims to:

  • Predict asset type, material, and structural features
  • Estimate size-related attributes such as height or length ranges
  • Handle challenges such as multiple images per asset and varying image quality

1.1.5 Final Data Product

The final product will be a machine learning pipeline that takes infrastructure images as input and outputs predicted attribute values in a structured format. The pipeline will include confidence scores to help identify uncertain predictions and support review by BC Parks staff. This system is intended to integrate with existing workflows and improve the overall reliability of the asset management database.
